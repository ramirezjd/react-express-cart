# Cart Server for test V-0.1

This is an attempt of a web cart server

## Getting Started
### Requirements
* Node V. 18.16.0
* Npm V. 9.5.1
### Installation
In order to install and run this project you sould execute the following commands:

* npm install
* npm run dev
